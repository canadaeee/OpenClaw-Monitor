# Observability Design - OpenClaw Monitor

## Metrics
- agent_online_count
- task_running_count
- token_consumed_per_model
- token_consumed_per_supplier
- node_offline_count

## Logs
- event streams from Gateway and Nodes
- formatted JSON logs for parsing

## Traces
- Correlate events belonging to the same task
- Assign trace_id to each task lifecycle
- Store trace data for deeper investigation
