# Terms & Definitions - OpenClaw Monitor

- **Agent**: OpenClaw 中执行任务的工作实例。
- **Node**: 计算执行单元，如 Windows 执行节点。
- **Task**: 用户或系统发起的操作请求。
- **Session**: 一个交互或对话的生命周期。
- **Token**: 模型调用时消耗的算力单位。
- **Model Supplier**: 模型提供者，如 OpenAI、Gemini 等。
- **Heartbeat**: agent 定时发送的在线信号。
- **Event Stream**: 按时间序列记录的事件日志。
- **MVP**: Minimum Viable Product（最小可用产品）。
